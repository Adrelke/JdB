Unit 2: CSS (6 jours)

DAY 6: CSS SETUP & SELECTORS

Journée découverte CSS, de son utilité, fonction, etc. Découverte des "tags" CSS et selectors (tag/class/id/!important).
Pour link une page CSS a un HMTL il faut utiliser le "link" suivant:

  • href = chemin du CSS file
  • type = text/css
  • rel = stylesheet

Qui donne au final la ligne de code : link href="./style.css" type="text/css" rel="stylesheet"
Beaucoup d'informations à retenir aujourd'hui.
